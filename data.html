Data API

used to retrieve information from Sensorflare.

List all available Resources

GET /api/resource

Returns a JSONArray with all the IoT Resources owned by the user and their names when available.

List All Available Dashboard

GET /api/dashboard

Returns a JSONArray with all the Dashboard created by the user.

List All Available Applications

GET /api/application

Returns a JSONArray with all the Applications deployed by the user.

List All Resources of a Dashboard

GET /api/dashboard/{DASHBOARD_NAME}/resource

Returns a JSONArray with all the IoT Resources added to the specified Dashboard.

List All Applications of a Dashboard

GET /api/dashboard/{DASHBOARD_NAME}/application

Returns a JSONArray with all the IoT Resources added to the specified Dashboard.

Get Resource Description

GET /api/resource/{RESOURCE_URI}/description

Returns a JSONArray with all the Applications deployed by the user.

Get Latest Summary of Resource

GET /api/resource/{RESOURCE_URI}/report/latest

Returns a JSONObject with the latest information concerning the IoT Resource.
