Sensorflare API reference: Schedules

Download the Sensorflare Library as zip file Download the Sensorflare Library as tar.gz file

Dashboards API

used to retrieve information about user schedules.

Add a Schedule to the system

GET /api/schedule

Returns a JSONArray with all the information about the Schedule added to the system.

Lists all existing schedules

GET /api/schedule

Returns a JSONArray with all the Schedules for a specific user.

Delete a Schedule

GET /api/schedule/{SCHEDULE_ID}

Returns {"status":"OK","code":200} if the specified Schedule was deleted successfully, otherwise returns "error".
