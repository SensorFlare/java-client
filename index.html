REST API

Sensorflare offers a REST API. Rest means that the URLs used are properly describing the actions and interactions performed. An overall categorization of the different interactions can be achieved using the following sections:

  • Authentication
    • Sensoflare uses Basic HTTP authentication to provide access to private resources.
  • Sense
    • It is used to retrieve information about sensors and the state of actuators.
  • Control
    • It is used to interact with IoT and Web resources.
  • Management
    • It is used to manage permissions, actions and resources.
