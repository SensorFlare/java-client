Dashboards API

It is used to retrieve information about user dashboards.


List All Available Dashboards

GET /api/dashboards

Returns a JSONArray with all the Dashboard created by the user.

Example:

{
                        dashboards:[
                        {id:1,name:"Dashboard1 Name"},
                        {id:2,name:"Dashboard2 Name"},
                        ...
                        ]
                        }

List all Components of a Dashboard

GET /api/dashboard/{dashboardId}

Returns a JSONArray with all the components of the specified Dashboard.

Example:

{
  id: 12345,
  name: "dashboardName",
  longtitude: 0,
  latitude: 0,
  tariff: null,
  featuredResource: null,
  parentDashboard: null,
  open: false
}

Set Location for a Dashboard

GET api/dashboard/{dashboardId}/location/{longtitude}/{latitude}

Returns a JSONArray with information about the specified dashboard if the location is set successfully, otherwise returns "error".

Example:

{
  id: 12345,
  name: "dashboardName",
  longtitude: 0,
  latitude: 0,
  tariff: null,
  featuredResource: null,
  parentDashboard: null,
  open: false
}

Create Dashboard

GET /api/dashboard/create/{dashboardName}

Returns a JSONArray with all the information about the created Dashboard.

Example:

{
  id: 12345,
  name: "dashboardName",
  longtitude: 0,
  latitude: 0,
  tariff: null,
  featuredResource: null,
  parentDashboard: null,
  open: false
}

Delete Dashboard

GET /api/dashboard/delete/{dashboardId}

Returns {"status":"Ok","code":200} if the specified Dashboard is deleted successfully, otherwise returns "error".

Add Resource to Dashboard

GET /api/dashboard/{dashbordId}/add/resource/**

Returns the JSON description of the Dashboard if the specified IoT Resource was added successfully, otherwise returns {"status":"Not Found","code":404} or {"status":"Forbidden","code":403}.

Disconnect Resource from Dashboard

GET /api/dashboard/{siteId}/disconnect/**

Returns {"status":"Ok","code":200} if the specified Resource is disconnected from Dashboard, otherwise returns "error".

Feature Resource in Dashboard

GET /api/dashboard/{siteId}/featured/**

Returns {"status":"Ok","code":200} if the specified Resource is featured in Dashboard, otherwise returns "error".

List All Resources of a Dashboard

GET /api/dashboard/{dashboardName}/resource

Returns a JSONArray with all the IoT Resources added to the specified Dashboard.

Example:

[
  "resourceName",
  ...
 ]
