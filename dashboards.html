Dashboards API

used to retrieve information about user dashboards.

List All Available Dashboard

GET /api/dashboard

Returns a JSONArray with all the Dashboard created by the user.

Create Dashboard

GET /api/dashboard/create

Returns a JSONArray with all the information about the created Dashboard.

Set Location For A Dashboard

GET /api/dashboard/{DASHBOARD_ID}/location

Returns {"status":"OK","code":200} if the location is set successfully, otherwise returns "error".

Add Resource To Dashboard 

GET /api/dashboard/add/resource

Returns {"status":"OK","code":200} if the specified IoT Resource is added successfully, otherwise returns "error".

Add Application To Dashboard

GET /api/dashboard/add/application

Returns {"status":"OK","code":200} if the specified Application is added successfully, otherwise returns "error".

Remove Application From Dashboard

GET /api/dashboard/remove/application

Returns {"status":"OK","code":200} if the specified Application is removed successfully, otherwise returns "error".

List all Components A Dashboard

GET /api/dashboard/{DASHBOARD_ID}

Returns a JSONArray with all the components of the specified Dashboard.

Delete Dashboard

GET /api/dashboard/delete

Returns {"status":"OK","code":200} if the specified Dashboard was deleted successfully, otherwise returns "error".

Dashboard Disconnect

GET /api/dashboard/{siteId}/disconnect/**

Dashboard Featured

GET /api/dashboard/{siteId}/featured/**

List All Resources of a Dashboard

GET /api/dashboard/{DASHBOARD_NAME}/resource

Returns a JSONArray with all the IoT Resources added to the specified Dashboard.

List All Applications of a Dashboard

GET /api/dashboard/{DASHBOARD_NAME}/application

Returns a JSONArray with all the Applications deployed by the user.
