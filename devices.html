Dashboards API

used to retrieve information about user devices.

List All The Gateways Of The User

GET /api/gateway

Returns a JSONArray with all the Gateways of the user.

Add Property To Gateway

GET /api/gateway/**

Returns {"status":"OK","code":200} if the Property is added successfully to the Gateway.

View Information Of A Gateway

GET /api/gateway/**

Returns a JSONArray with all the information about the specified Gateway.

List All The Devices Of The User

GET /api/device

Returns a JSONArray with all the Devices of the user.
